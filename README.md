# 北京訪問先ドシエ v4

北京旅行で回りたい店舗・市場のリストを管理する、単一HTMLファイルの旅行ツールです。

## 機能

- **訪問先カード**：店名（日中）、住所、ピンイン、最寄駅、周回パターン（A東部/B西部/C南部/補 亦庄）
- **高徳地図・Apple マップ・百度地図へのワンタップ起動**：カード内のボタンからナビアプリに直結
- **住所コピー**：タクシー運転手に見せるための中文住所をクリップボードにコピー
- **カテゴリ・パターン・テキスト絞り込み**
- **地下鉄経路図**（SVG）：4パターン分の周回ルートを図示
- **追加・編集・削除**：画面上のフォームから操作可能
- **エクスポート・インポート**：JSON形式でデータの持ち運び
- **地図表示**（オプション）：GitHub Pages 等 HTTPS 経由で開いた場合のみ表示。ローカルファイルや中国本土ではフォールバックメッセージに切り替わり、エラーは出ない

## 使い方

### GitHub Pages で公開する場合

1. このリポジトリを GitHub に push する（private 推奨）
2. Settings → Pages → Source で `main` ブランチ `/root` を選択
3. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開される

### ローカルファイルとして使う場合

`index.html` をダウンロードし、ブラウザで直接開く。地図は表示されないがカード・経路図・追加削除機能は全て動作する。

### スマートフォンで使う場合

- **日本で準備**：GitHub Pages の URL をブックマーク or `index.html` をローカル保存
- **中国本土**：GitHub Pages は VPN なしではアクセス不安定。事前にローカル保存した `index.html` をブラウザで開く
- **ナビの流れ**：カードの「高徳で開く」ボタンをタップ → Amap アプリが起動 → ナビ開始

### データの管理

| 操作 | 方法 |
|---|---|
| 追加 | 「＋ 新規追加」ボタン → フォーム入力 → 保存 |
| 編集 | カードの「編集」ボタン |
| 削除 | カードの「削除」ボタン |
| 別端末へ移行 | 「エクスポート」でJSON保存 → 別端末で「インポート」 |
| 初期状態に戻す | 「初期データに戻す」ボタン |

データはブラウザの `localStorage` に保存されます。同じブラウザで再度開けば変更が維持されます。

## 事前にインストールするアプリ

| アプリ | 用途 |
|---|---|
| 高徳地図（Amap） | メインのナビ。App Store / Google Play で「Amap」検索 |
| 百度地図 | Amap で見つからない場合のバックアップ |
| Apple マップ | iPhone 標準。中国国内でも動作 |

## 注意事項

- 地図のピン位置は WGS-84 座標系による概算値です。中国本土の地図サービス（高徳・百度・Apple マップ）は GCJ-02 座標系のため、数十〜数百m のズレが発生します。**ナビは必ず「高徳で開く」ボタンから**行ってください。
- NITORI 遠洋未来広場店は 2024年1月閉鎖済のため、凱德MALL太陽宮店に差し替え済みです。
- 鼎好大厦・中関村科貿電子城は再開発による業態縮小が進行中です。訪問前に大衆点評で現況を確認してください。
